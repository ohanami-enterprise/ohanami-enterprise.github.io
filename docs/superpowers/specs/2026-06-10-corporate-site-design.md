## 3. デザイン方向性

**和モダン × 猫2匹モチーフ**

| 要素 | 指定 |
|------|------|
| 背景 | 生成り（washi）`#faf5ee` |
| 本文テキスト | 墨色 `#2a221e` / 補助 `#6b5f54` |
| アクセント | 茶系 `#a98a6a`〜`#d99b5f` |
| 見出し書体 | 明朝体（Hiragino Mincho ProN / Yu Mincho 系。Webフォントは必要に応じ検討） |
| トーン | 落ち着いた、温かみのある和の質感。テンプレ然とした既製感を避ける |
| レスポンシブ | スマホ／PC 両対応 |

### 猫2匹モチーフ（SVGで実装・確定デザイン）

Hero の右下に2匹を並べて配置。実在の飼い猫がモデル。

- **大きい方（白茶・牛柄）**
  - 白ベースに茶のブチが点在（牛柄）。片耳と片目まわりに茶ブチ。
  - **短く太い鍵尻尾**（先がカクッと折れ、太さは均一、体に密着。折れた先は茶）。
- **小さい方（三毛・黒ヘルメット）**
  - 白ベースの体に黒・茶のブチ（三毛）。しっぽは黒。
  - **頭は天辺まで黒**（耳も黒）、目から下の顔のみ白く抜く（ヘルメットを被ったような配色）。

ブラウザブレストの確定版モック：`.superpowers/brainstorm/.../content/cats-real-v9.html`
