# ohanami-enterprise.github.io

合同会社おはなみエンタープライズ コーポレートサイト。

- 公開URL: https://ohanami-enterprise.github.io
- 構築: Hugo（テーマ無し・独自レイアウト）
- デプロイ: `main` への push で GitHub Actions が自動公開

## ローカル開発

```bash
hugo server
# http://localhost:1313
```
